# PGAI_VoiceBot
 PGAI Engineering Challenge for Joseph Yoon

If running on Linux or on a machine that prevents full Python access, type the following commands into the terminal in the folder PGAI_Challenge.

0. 
    For macOS/Linux: Type python3 -m venv venv 
    For Windows: Type python -m venv venv


If a venv is not necessary:
1. 
    For macOS/Linux: Type python3 run.py
    For Windows or venv: Type python run.py


Executing run.py will allow for telephony_agent.py to run, and after a few seconds, call.py will initailize the patient agent and connect it to the phone.
